Cloud API (Recommended)

OpenRouter, Anthropic, or OpenAI — best quality, no hardware needed

▶

OpenRouter (easiest, free tier available)

  1. Sign up at openrouter.ai
  2. Go to Keys → Create Key
  3. Paste below and click Save

Free model: nvidia/nemotron-3-super-120b-a12b:free (262K context). $10 credit unlocks 1000 req/day.

Anthropic (Claude)

  1. Get an API key at console.anthropic.com
  2. Paste below and click Save

OpenAI

  1. Get an API key at platform.openai.com
  2. Paste below and click Save
💻

Local Model (Private, Free)

Ollama, LM Studio, or llama.cpp — runs on your own hardware, no API costs

▶

Ollama (simplest local option)

  1. Install from ollama.ai
  2. Run: ollama pull llama3.1
  3. Ollama runs on port 11434 by default

LM Studio / llama.cpp

  1. Start your server (default port 1234 for LM Studio)
  2. Enter the URL below
⚡

Hybrid (Cloud + Local Fallback)

Cloud for quality, local for privacy/offline — automatic failover

▶

Set up a cloud provider above as primary, then add a local model as fallback. Syntaur tries the primary first and falls back automatically if it fails or is unreachable.

  • ✓ Music playback through phone speakers, AirPods, or AirPlay

  • ✓ Voice journal capture from anywhere
