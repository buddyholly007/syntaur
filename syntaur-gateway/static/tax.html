Accuracy tips

The scanner uses AI vision to read your documents. For best results:

• Upload clear, high-resolution scans (photos of paper documents work too)

• PDFs are automatically converted to high-res images for better reading

• If you see frequent errors, switching to a more capable model in Settings can significantly improve accuracy.

Free models (OpenRouter):

• NVIDIA Nemotron Nano VL — #1 on OCR benchmarks, purpose-built for documents (current default)

• Google Gemma 4 31B — strong vision with 262K context for large documents

• Google Gemma 4 26B — lighter alternative, nearly as accurate

Self-hosted (free, private, offline):

• NVIDIA Nemotron Nano 12B VL — same model as cloud default, runs locally on any GPU with 10+ GB VRAM. Q4_K_M quantization needs ~9 GB. Download GGUF from HuggingFace, run via llama.cpp. Cannot run alongside a large chat model — swap models when scanning.

Setup: llama-server -m Nemotron-Nano-12B-v2-VL-Q4_K_M.gguf --mmproj mmproj-BF16.gguf --port 1237 -ngl 99

Paid cloud (highest accuracy):

• Anthropic Claude Sonnet — best overall accuracy for complex tax documents

• OpenAI GPT-4o — excellent OCR, especially for handwritten notes

• Qwen3 VL 235B — top-tier vision, best value at scale

• Always verify extracted values against your original documents
